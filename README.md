# Sui Staker

A simple web UI for staking SUI tokens to active and pending validators on the Sui network.

## Features

- Connect with Sui wallets (Slush etc.)
- Select from active validators or enter a custom pending validator address
- View validator info
- Stake SUI to validators
- Switch between Mainnet, Testnet, or custom RPC endpoints
- Transaction confirmation with explorer links

## Getting Started

### Prerequisites

- Node.js 18+
- A Sui wallet browser extension

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

The production build will be output to the `dist/` directory.

## Deployment

This is a static web application with no backend required. Deploy the `dist/` folder to any static hosting service.
